# shaker
## Description
Shakes the mouse and presses shift every 3 minutes to keep the screen awake. 

## Requirements
requires python 3+ and pyautogui

## Install instructions
 1. Install python and add to path
 2. install pyautogui
 ```
 python -m pip install pyautogui
 ```
 

## Example Invocation
 ```
python ./shake.py
 ```
